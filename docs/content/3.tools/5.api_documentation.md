We use [Swagger-PHP](https://zircote.github.io/swagger-php/) for annotations and generating the OpenAPI specification.

## Usage

::steps
  ### Annotate Code
  Add Swagger-PHP annotations to your classes and methods.

  ### Generate Spec
  Run the following command to update [openapi/openapi.yml](https://github.com/TerrorSquad/php-booster/blob/main/booster/openapi/openapi.yml):
  ::code-group
    ```bash [Standard]
    composer generate-api-spec
    ```
    ```bash [DDEV]
    ddev composer generate-api-spec
    ```
  ::
::

## Automation

Git hooks automatically regenerate documentation when `openapi.yml` changes.
